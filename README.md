# Mobile-Phone-Details
The Mobile Phone Details Web App is an interactive frontend project designed to display detailed specifications and features of various mobile phones. Built using HTML, CSS, and JavaScript, this project focuses on providing a clean UI and dynamic interactivity for users exploring mobile devices.

**Key Features**

Mobile Details Display – Showcases specifications like model, price, RAM, storage, camera, and battery life.

Dynamic Interactions – Uses JavaScript to fetch, display, or toggle between phone details without refreshing the page.

Search & Filter Options (optional) – Allows users to quickly search for phones based on brand, price, or specifications.

Responsive Design – Fully optimized for desktop, tablet, and mobile using CSS flexbox, grid, and media queries.

Attractive UI – Uses modern CSS styling for a clean, professional, and user-friendly interface.

**Technologies Used**

HTML5 → For structuring the content and layout of mobile phone details.

CSS3 → For styling, responsive design, and creating an attractive UI.

JavaScript (ES6) → For dynamic interactions, event handling, and updating content seamlessly.
